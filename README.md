# 🔧 Web ToolBox - 程序员在线工具箱

一个简洁、高效、安全的在线工具集合网站，基于 Vue 3 + TypeScript + Vite 构建。所有工具逻辑均在浏览器本地运行，保护您的数据隐私。

## ✨ 特性

- 🎨 **现代化 UI 设计**：基于 Tailwind CSS + DaisyUI，界面美观且支持暗色/亮色模式。
- 📱 **响应式布局**：完美适配 PC、平板和移动端设备。
- � **本地处理**：数据无需上传服务器，所有解析和转换逻辑均在本地完成，确保隐私安全。
- 🚀 **极致体验**：支持路由懒加载、大文件分页处理，交互流畅。
- 🔍 **SEO 优化**：针对搜索引擎进行了深度优化，包含动态 Meta 标签、Sitemap 和 Robots.txt。

## 🛠️ 技术栈

- **框架**: Vue 3 (Composition API)
- **语言**: TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS + DaisyUI
- **部署**: Cloudflare Pages (推荐)

## 📦 安装与运行

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 生产构建
```bash
npm run build
```

## 🎯 已完成工具

### 文本工具
- **JSON 格式化** ✅：支持缩进调整、语法高亮和一键复制。
- **时间戳转换** ✅：支持 Unix 时间戳与日期格式（YYYY-MM-DD HH:mm:ss）互相转换。
- **Base64 编解码** ✅：支持文本与 Base64 格式的互相转换。
- **CSV 解析器** ✅：
  - 支持多文件同时解析。
  - 大文件自动分页展示，解决渲染卡顿。
  - **多格式 JSON 导出**：支持对象数组、二维数组、仅提取值等多种导出模式。

### 图片工具
- **图片九宫格切分** ✅：一键将图片均匀切分为 3x3 的九张图，支持批量下载，适配朋友圈、微博等九宫格发布。
- **图片压缩** ✅：在线压缩图片体积，支持调整质量和尺寸。
- **图片 Base64** ✅：支持将本地图片或图片 URL 转换为 Base64 编码。
- **Favicon 在线制作** ✅：快速将图片转换为多种尺寸的浏览器图标（16x16, 32x32 等）。
- **九宫格文字** ✅：输入文字自动生成九宫格切片图片，支持自定义背景、颜色、边框及圆角，适配社交平台发布。

### 转换工具
- **颜色转换器** ✅：支持 HEX、RGB、HSL 格式互转，带颜色预览。

### 生成工具
- **密码生成器** ✅：支持自定义长度、字符类型，生成强安全随机密码。
